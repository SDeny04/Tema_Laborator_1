# Tema_Laborator_1

Magazin online componente PC
Scopul aplicației este:

-gestionarea stocurilor
  -adăuga produse noi
  -modifica prețul sau specificațiile
  -șterge produse
  -actualiza cantitatea în stoc
  
-administrarea utilizatorilor
  -creare cont și autentificare
  -modificare date personale
  -diferențiere roluri (client / admin)
  -posibilitatea blocării sau ștergerii unui utilizator (de către admin)
  
-procesarea comenzilor
  -adăuga produse în coș
  -modifica cantități
  -confirma comanda
  
-păstrarea unui istoric al tranzacțiilor
  -data comenzii
  -produsele achiziționate
  -valoarea totală
  -statusul comenzii
